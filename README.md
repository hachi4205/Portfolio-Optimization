# Portfolio Optimization using Linear Algebra - Vietnamese Stock Market

Final project for Linear Algebra course, applying PCA, Multiple Linear Regression, 
and Sharpe Ratio optimization on HOSE blue-chip stocks.

## Stocks Used
VNM, VIC, VHM, FPT, HPG, MWG, VCB, MBB

## Data Period
2022-01-01 to 2025-05-01

## Tech Stack
- Python 3.13
- pandas, numpy, scipy, matplotlib, scikit-learn
- vnstock (Vietnamese stock data API)
